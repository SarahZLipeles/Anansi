Create function by creating an object that has 2 methods ('start' and 'receive')


The 'start' function is triggered when you click on a node

The 'receive' function is triggered in response to all of the actions performed on the nodes


The following are the supported functions:

  • attack(sourceId, targetId)
  • reinforce(nodeId)

Neither attack() nor receive() have return values. However, on completion, your crawler triggers your "receive" function with an node containing a nodeId and either remaining health or a list of the connected nodeIds in the following format

{id: String, health: Number}

{id: String, links: [nodeId]}

The following functions returns booleans

  • isEnemy(nodeId)
  • isFriend(nodeId)
  • isNeutral(nodeId)
